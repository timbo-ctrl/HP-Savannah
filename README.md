# HP Savannahs

The homepage for HP Savannahs — a second-generation cattery placing high-percentage (70%+) F1 Savannahs by application only.

## Stack

Single-file static site. No build step required.

- `index.html` — all markup, styles, and scripts in one file
- `assets/` — logo, imagery, and reference material

## Running locally

Open `index.html` directly in a browser, or serve the directory:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Design

- **Palette** — Chicago Bulls (red / black / bone)
- **Typography** — Anton (display), Instrument Serif (editorial italic), Archivo (body), JetBrains Mono (meta)
- **Direction** — Editorial magazine luxury. Richard Mille / Off-White editorial confidence, not quiet luxury.

## Structure

1. Hero — massive display type with cinematic imagery
2. Ticker — brand claims in motion
3. Manifesto — the anonymous-breeder pull quote
4. The Breed — "What is High Percentage?" with stats
5. The Cats — current litter gallery
6. Split Quote — full-bleed editorial moment
7. The Process — application-based placement flow
8. Journal — editorial depth
9. CTA — Request Consideration
10. Footer
